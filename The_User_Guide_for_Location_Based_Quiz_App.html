1. Start the Location_Based Quiz App

Open the software Bitvise SSH Client 6.31 and type the following code in the terminal to connect to the right folder:

cd /home/studentuser/code/server/uczlwj0

Then type the following code to start the httpServer:

node httpServer.js &

Next, you should connect to another folder by typing:

cd /home/studentuser/code/quiz/uczlwj0

And type the following code to start the PhoneGap:

phonegap serve

Finally, type the following website in the browser explorer, you can see the interface of the Question Setting App and start to use it.

http://developer.cege.ucl.ac.uk:31275/

2. Track the location of the user

You can find a drop-down list on the bottom of the App. You can track your location by pressing the button of "Track Location".

Then you can see a marker is placed on the map which represents your position.

Also your coordinates of your position is shown on the bottom of the App.

3. Answer the question

Once you are close to the location within a certain distance, a quiz will be shown on the sidebar.

This is a question concerned with the location.

You can select the answer by clicking the radio button, and then submit your answer.

Your phone ID and your answer will be stored in the App.

4. Check the answer

You can click the "Show Correct Answer" button to check whether your answer is right or wrong.

If you are wrong, the App will tell you the right answer.

5. Log out the App

It would be better if you can close the PhoneGap by typing the following code in the terminal of Bitvise SSH Client 6.31:

“Ctrl” + C

Do not forget to close httpServer either:

fg 1

“Ctrl” + C
